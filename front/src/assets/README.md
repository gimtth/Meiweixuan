# 图片资源说明

本目录用于存放网上订餐系统前端所需的图片资源。您需要添加以下几个关键图片文件:

## 必需图片资源

1. **restaurant.jpg** - 餐厅环境图片，用于首页"关于我们"部分
2. **banner1.jpg**, **banner2.jpg**, **banner3.jpg** - 首页轮播图图片
3. **default-dish.jpg** - 默认菜品图片，用于菜品无图片时显示

## 如何添加图片

您可以通过以下方式添加所需图片：

1. 准备相应尺寸的图片文件（推荐尺寸如下）：
   - 轮播图: 1200px × 400px
   - 餐厅环境图: 600px × 400px
   - 默认菜品图: 300px × 200px

2. 将图片文件复制到此目录下，并确保文件名与上述一致

3. 重新启动开发服务器或刷新页面查看效果

## 临时解决方案

如果没有立即可用的图片资源，系统已配置默认占位内容，可以正常使用。
您也可以使用在线占位图片服务，比如：

- 轮播图: `https://via.placeholder.com/1200x400?text=餐厅美食`
- 餐厅图: `https://via.placeholder.com/600x400?text=餐厅环境`
- 菜品图: `https://via.placeholder.com/300x200?text=菜品图片`
